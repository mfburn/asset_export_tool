# asset_export_tool
FPX export helper for Blender

Aддон для Blender, упрощающий экспорт ассетов в FBX. Позволяет экспортировать иерархию объектов, автоматически проверяет scale и отображает (shared mesh) дубликаты перед экспортом.

Установка:

Скачай или клонируй репозиторий:

git clone https://github.com/yourname/asset-export-tool.git
Открой Blender

Перейди в:
Edit → Preferences → Add-ons

Нажми Install…

Выбери файл:
addon/asset_export_tool.py

Включи аддон галочкой Asset Export Tool
